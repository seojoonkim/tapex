# TAPEX 4대 관점 콘텐츠 리뷰

## 1. 마케팅 카피라이터 관점

### 설득력 점수
- Hero: 7/10 — 깔끔하지만 감정적 훅이 약함
- Authority Quotes: 8/10 — 데이터 활용 훌륭
- Why TAPEX: 7/10 — 논리적이나 마지막 펀치 약함
- Persona: 6/10 — quotes가 약간 작위적
- Enterprise: 5/10 — 너무 짧고 구체성 부족
- CTA: 6/10 — 긴급성(urgency) 없음

### 핵심 문제
1. "시험 접수하기" — 가장 중요한 CTA가 너무 무미건조
2. Hero 서브타이틀이 기능 나열에 그침
3. "지금 바로 시작하세요" — 모든 사이트에서 쓰는 클리셰
4. 페르소나 quotes가 너무 길고 설명적

### AI가 쓴 것 같은 표현
- "글로벌 AI 활용 역량 인증 표준" — 브로셔체
- "객관적으로 측정하는 글로벌 표준은 지금까지 없었습니다" — 과장
- "먼저 준비한 사람이 기회를 잡습니다" — 자기계발서 클리셰

## 2. 취준생/직장인 관점

### 동기부여 되는가?
- 연봉 56% 데이터 → 강력하게 와닿음 ✅
- 6배 생산성 격차 → 와닿음 ✅
- "Bronze(600+)는 객관적 수치로 차별화" → 뭘 준비해야 하는지 모름 ❌
- 시험 난이도, 소요시간, 합격률 정보 없음 ❌

### 공허한 표현
- "AI를 전략적으로 설계하고 조율하는 최상위 역량" — 구체적으로 뭘?
- "AI를 본격적으로 배우고 익혀가는 학습 단계" — Novice에 대한 위로 불필요

### 추가 필요 FAQ
- 시험 시간은 얼마나?
- 어디서 보나? (온라인? 오프라인?)
- 준비는 어떻게?
- ChatGPT를 직접 쓰면서 시험 보나?

## 3. 기업 HR 관점

### Enterprise 섹션 평가
- 현재: 2문단 + 체크리스트 3개 → 너무 얕음
- "69.2%가 채용 시 AI 역량을 고려" — 좋은 데이터지만 출처 부족
- ROI 수치 없음 (도입 시 채용 효율 몇% 향상 등)
- 도입 사례 없음
- 가격 정보 없음 (기업용)

### 필요한 추가 요소
- 단체 응시 할인 구조
- 팀별 리포트 예시
- "잡코리아·사람인 연동"의 구체적 화면/프로세스
- 보안/데이터 처리 정보

## 4. 글로벌 자격증 경쟁력 관점

### 권위도 평가
- TOEIC과의 비교 구조는 효과적 ✅
- 한국경제신문 인증 → 국내에선 좋지만 글로벌 신뢰도 한계
- "160개국" → 온라인이라 가능한 건데, 실제 응시자 분포 데이터 없음
- 파트너사가 3개뿐 (한국경제, HASHED, bloomingbit) → 국제 기관 없음

### 영문 표현
- "Test of AI Prompt EXpertise" — EX 대문자 의도는 이해하나 어색
- 영어 quotes는 적절
- 글로벌 사이트에서 쓰려면 영문 페이지 필요

### 신뢰도 향상 방안
- 국제 인증 기관과의 파트너십 필요
- 글로벌 기업 도입 사례
- 영문 성적표/인증서 샘플
